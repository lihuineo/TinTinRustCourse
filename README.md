第一课作业题目：  
一、添加一个一层子模块，循环打印从’a’~’Z’ 之间的所有字符  
二、添加一个二层子模块，循环打印从’A’~’z’ 之间的所有字符  
三、使用Cargo编译运行此工程  

第一课作业解答：  
一、第一层子模块char_helper, 函数a_to_z()实现循环打印从'a'~'Z' 之间的所有字符    
二、第二层子模块util/data_helper, 函数a_to_z()实现循环打印从'A'~'z'之间的所有字符
